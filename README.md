# ✨ Overview

PersistenceAI is a fully native AI assistant developed entirely with **SwiftUI**.

The application is designed to provide a smooth, responsive, and elegant AI chatting experience while embracing Apple's latest design language. It combines modern navigation, configurable AI settings, conversation management, image support, and beautiful **Glass UI** elements into a lightweight application that feels at home across Apple platforms.

The project focuses on:

- Native SwiftUI architecture
- Modern Glass UI
- Fast performance
- Configurable AI
- Clean user experience
- Cross-platform Apple support
